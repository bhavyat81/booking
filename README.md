## Email Notifications (Auto-Send Booking Emails)

When a booking is submitted, the app automatically sends the booking details to **bhavyat81@gmail.com** via a Google Apps Script webhook — no email app is opened and no user interaction is required.

- Email is sent automatically via Google Apps Script webhook
- Booking details are sent to **bhavyat81@gmail.com**
- No setup required — it just works!
